# Freelancer Portfolio Bootstrap Version

This version implements the same Studio/Freelancer Portfolio mockup using Bootstrap 5 grid, navbar, responsive utilities, and components.

## Run

Open `index.html` in a browser with an internet connection so the Bootstrap CDN files can load.

## Structure

- `index.html`: Bootstrap-based page markup
- `styles.css`: custom styles for the provided mockup
- `resources/`: provided images and logos

Bootstrap is used through CDN links. No additional local build step is required.
